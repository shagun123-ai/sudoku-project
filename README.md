Sudoku Solver (C)

A simple Sudoku solver built using the backtracking algorithm.
It checks whether a number can be placed in a specific position and can solve the full puzzle logically.

Features
Validates moves
Solves Sudoku using backtracking
Beginner-friendly C implementation

What I Learned
Logical thinking
Backtracking
Strengthening C basics

How to Run
gcc sudoku.c -o sudoku
./sudoku
